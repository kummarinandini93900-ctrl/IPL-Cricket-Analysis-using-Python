# IPL-Cricket-Analysis-using-Python
Analyzed IPL match and ball-by-ball data using Python, Pandas, and Matplotlib to identify team performance, player statistics, venue trends, and scoring patterns.
## Tools Used
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## Dataset
- matches.csv
- deliveries.csv

## Analysis Performed
1. Top Winning Teams
2. Top Players of the Match
3. Top IPL Venues
4. Matches Per Season
5. Most Runs by Batsmen
6. Most Wickets by Bowlers
7. Runs Per Over

## Conclusion
- Mumbai Indians emerged as one of the most successful IPL teams.
- Run scoring increases during death overs.
- Certain venues hosted more matches than others.
